# Git Weekly Timesheet Generator

這個小工具可以幫你自動掃描資料夾內的 Git 專案，找出你過去幾天內的 commits，並自動幫你組合整理成含有表格的「Markdown 格式 (.md)」，方便你直接圈選複製並用 Excel 或 Google 表格貼上填寫工時表！

## 🚀 如何分享與使用？

只需將同一個資料夾中的 `weekly_report.sh` 檔案傳給你的同事/朋友即可。這是一支不需要安裝任何軟體（如 Node.js 或 Python）的原生 Mac / Linux 腳本。

### 基本用法 (掃描當前目錄)

1. 將 `weekly_report.sh` 放到你存放專案的資料夾底下（例如 `~/Desktop/Your_Projects/`）。
2. 打開終端機 (Terminal)。
3. 給予腳本執行權限 (只需要做一次)：
   ```bash
   chmod +x weekly_report.sh
   ```
4. 執行腳本：
   ```bash
   ./weekly_report.sh
   ```
   它預設會去掃描 **當前位置底下所有資料夾**，並過濾出 **「你綁在 Git 上的帳號本身」所做出的所有 commit**，幫你整理過去 7 天的成果！

### 進階用法 (自訂路徑與天數)

你可以直接在後面加上「你想掃描的資料夾路徑」以及「天數」。
例如，掃描上一層資料夾，並抓取過去 14 天的資料：
```bash
./weekly_report.sh ../ 14
```

## 📊 如何使用產生的報告？
程式執行完畢後，會在這個檔案旁邊默默整理出一份例如 `Weekly_Report_20260417.md` 的 Markdown 文字檔。
1. **任何文字編輯器預覽**：你可以使用 VS Code 或是 Markdown 檢視軟體打開它。預覽後，你可以發現它是一個排版完美的表格。
2. **無縫複製貼上**：直接在預覽畫面「全選表格」，按 `Ctrl/Cmd + C` 複製，接著到你的 Google 表格隨便點一格按貼上，它就會自動化分成完美的一格一格囉！

## 💡 特色
- 🕒 **留白工時**：自動去除 Git 判斷不出的真實時數，為你預留了「週一至週五」的空格，方便貼上直接填寫。
- ✅ **中文字元保護**：採用 UTF-8 BOM 格式建立，Excel 不會亂碼。
- 👥 **自動偵測姓名**：會自己抓取使用者的 `git config user.name`，不用額外到腳本裡設定姓名。
